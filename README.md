# QueueManagementSystem
 Queue Management System - C++

A professional Queue Management System built with C++ and Object-Oriented Programming principles. This system simulates real-world queue operations like those used in banks, hospitals, and service centers.

##  Features

### Ticket Management
- Automatic Ticket Numbering with custom prefixes (e.g., "A001", "B002")
- Real-time Timestamping for each ticket issue time
- Unique Ticket Identification combining prefix and sequential numbers

### Queue Operations
- Add Clients to queue with automatic ticket generation
- Serve Next Client with proper queue management
- Waiting Clients Tracking in real-time
- Queue Statistics monitoring

### Service Time Management
- Configurable Average Service Time per queue
- Expected Wait Time Calculation for each client
- Smart Time Estimation based on queue length and service time

### Multiple Display Modes
- Right-to-Left (RTL) queue visualization
- Left-to-Right (LTR) queue visualization  
- Formatted Ticket Printing with full details
- Comprehensive Queue Reports

##  Technical Stack

- C++17 - Core programming language
- Object-Oriented Programming - Clean architecture
- STL Containers - queue and stack for data management
- Custom Date Library - For timestamp functionality
- Modular Design - Separated concerns and reusable components
